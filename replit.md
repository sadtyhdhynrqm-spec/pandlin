# مشروع بوت SNFOR

## معلومات المشروع
- **الاسم**: SNFOR Bot
- **النوع**: بوت فيسبوك
- **المطور**: حمودي سان / سينكو
- **اللغة**: Node.js

## التعديلات الأخيرة
- تغيير أسماء مجلدات الأوامر من إنجليزي إلى عربي
- حذف الأوامر المكررة والزائدة للوصول لـ 80 أمر
- تزقرف قائمة الأوامر بتصميم جديد
- تسريع البوت: إيقاف listenTyping و updatePresence

## مجلدات الأوامر (العربية)
```
commands/
├── ذكاء/       (9 أوامر) - الذكاء الاصطناعي والصور
├── مطور/       (9 أوامر) - أوامر المطور فقط
├── مجموعة/     (7 أوامر) - إدارة المجموعات
├── اقتصاد/     (6 أوامر) - الاقتصاد والعملات
├── ترفيه/      (28 أمر)  - الترفيه والألعاب
├── وسائط/      (6 أوامر) - الميديا والوسائط
├── أدوات/      (12 أمر)  - الأدوات العامة
└── أخرى/       (3 أوامر) - أوامر متنوعة
```

## البنية الكاملة
```
├── commands/          # أوامر البوت (مجلدات عربية)
├── events/            # معالجات الأحداث
├── handler/handlers.js # معالج الأوامر الرئيسي
├── listen/listen.js   # مستمع الرسائل
├── helper/            # دوال مساعدة + ترجمة
├── logger/            # نظام السجلات
├── locales/           # ملفات الترجمة العربية
├── database/          # قاعدة البيانات
├── middleware/        # تحميل الأوامر والأحداث
└── KaguyaSetUp/       # إعدادات البوت
```

## الإعدادات المهمة
- **الملف**: `KaguyaSetUp/config.js`
- **البادئة**: `*`
- **listenTyping**: false (لتسريع البوت)
- **updatePresence**: false (لتسريع البوت)
