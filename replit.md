# مشروع بوت SNFOR

## 📝 معلومات المشروع
- **الاسم**: SNFOR Bot
- **النوع**: بوت فيسبوك متقدم
- **المطور**: حمودي سان 🇸🇩
- **اللغة**: عربي + إنجليزي

## ✅ الإصلاحات المطبقة

### 1. إصلاح نظام السجلات (Logger)
- **المشكلة**: استيراد `node-notifier` غير الضروري
- **الحل**: استبدله برسائل console آمنة

### 2. نظام الترجمة العربي (i18n)
- **الملف**: `locales/ar.json`
- **الملف**: `helper/translate.js`
- **الفائدة**: ترجمة موحدة لجميع الرسائل

### 3. تحديث معالج الأوامر
- استخدام نظام الترجمة في جميع الرسائل
- رسائل خطأ موحدة وعربية

## 📁 البنية الحالية
```
├── commands/          # جميع أوامر البوت
├── events/            # معالجات الأحداث
├── handler/           # معالج الأوامر الرئيسي
├── helper/            # دوال مساعدة + ترجمة
├── logger/            # نظام السجلات
├── locales/           # ملفات الترجمة العربية
├── database/          # قاعدة البيانات
└── KaguyaSetUp/       # إعدادات البوت
```

## 🎯 الخطوات التالية
- اختبار البوت والتأكد من عمل جميع الأوامر
- إضافة ترجمات إضافية حسب الحاجة
- تحسين الأداء وتحسين معالجة الأخطاء
